# 🥗 Smart Nutrition Deficiency Predictor

## 📌 About the Project

This project helps to predict possible nutrition deficiencies based on your diet and lifestyle.

It uses Machine Learning and a interactive web app built with Streamlit.

---

## 🚀 Features

* Predicts:

  * Iron deficiency
  * Vitamin B12 deficiency
  * Vitamin D deficiency
  * Calcium deficiency
* Easy to use interface
* Gives food suggestions
* Shows health tips

---

## 🛠 Technologies Used

* Python
* Streamlit
* NumPy
* Scikit-learn

---

## ▶️ How to Run

1. Install required libraries:

```bash
pip install streamlit
```

2. Run the app:

```bash
streamlit run Nutrition.py
```

---

## 📊 How It Works

* You enter your details (age, diet, lifestyle)
* The model checks your data
* It shows possible deficiencies
* It suggests foods to improve your health
